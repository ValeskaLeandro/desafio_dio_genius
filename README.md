## Jogo da memória - GENIUS

#### Objetivo: Criar uma aplicação que simule o jogo GENIUS. Onde as cores são sorteadas de maneira randômica e o jogador deve replicar a ordem em que forem apresentadas.

* Temas abordados:
    * flexbox;
    * display:grid;
    * grid-area;
    * Arrow functions;
    * alerts;
    * for/in;
    * setTimeout();
